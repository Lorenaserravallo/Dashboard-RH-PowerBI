# Dashboard-RH-PowerBI
Dashboard de RH desenvolvido no Power BI com tratamento de dados no Power Query, criação de indicadores em DAX (Contratações, Demissões, Funcionários Ativos e Turnover) e construção de visuais interativos como Sparkline, Funil, Rosca, Árvore Hierárquica e Tooltip personalizada para análise estratégica de pessoas.
